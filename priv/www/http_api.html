Emqttd Dashboard HTTP API

 

Introduction

Apart from this help page, all URIs will serve only resources of type application/json, and will require HTTP basic authentication. The default user is admin/admin.

The emqttd_dashboard plugin provide emqttd web management interface.The Dashboard include emqttd broker monitor, clients management, sessions management, topics management and so on. The Emqttd Dashboard Plugin Main Interface.

Reference

GET Path Description
X /api/stats The emqttd broker statistics of clients, topics, subscribers
X /api/node A list of nodes in the Emqttd broker cluster
X /api/metrics The emqttd broker metrics
X /api/listeners A list of all listeners in the emqttd broker
X /api/clients A list of all connected clients
X /api/session A list of all sessions the emqttd broker
X /api/topic A list of all topic in the emqttd broker
X /api/subscriber A list of all subscriber in the emqttd broker
X /api/users A list of all emqttd_dashboard management user
