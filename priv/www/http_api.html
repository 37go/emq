eMQTTd Dashboard HTTP API

Introduction

Apart from this help page, all URIs will serve only resources of type application/json, and will require HTTP basic authentication. The default user is admin/admin.

The emqttd_dashboard plugin provide eMQTTd web management interface.The Dashboard include eMQTT broker monitor, clients management, sessions management, topics management and so on. The eMQTTd Dashboard Plugin Main Interface.

Reference

GET Path Description
X /api/stats The eMQTT broker statistics of clients, topics, subscriptions
X /api/node A list of nodes in the eMQTT broker cluster
X /api/metrics The eMQTT broker metrics
X /api/listeners A list of all listeners in the eMQTT broker
X /api/clients A list of all connected clients
X /api/sessions A list of all sessions the eMQTT broker
X /api/topics A list of all topic in the eMQTT broker
X /api/routes A list of all route in the eMQTT broker
X /api/subscriptions A list of all subscriber in the eMQTT broker
X /api/users A list of all emqttd_dashboard management user
